How this AWS Website Works:

This website operates with AWS services, utilizing HTML, CSS, and JavaScript for the frontend. The backend infrastructure is powered by various AWS services:

  • Route 53: Provides domain registration and DNS routing.
  • ACM (AWS Certificate Manager): Manages SSL/TLS certificates for secure communication.
  • CloudFront: Acts as a content delivery network (CDN) to accelerate content delivery globally.
  • S3 (Simple Storage Service): Stores static assets such as HTML files, JavaScript files, CSS stylesheets, and images.
  • Lambda for Counting Views: Executes serverless functions in this case for retrieving and updating the count value from DynamoDB.
  • Lambda for Email Notification: Get data from dynamodb trigger lambda function to send message via email using SNS
  • DynamoDB: Provides a NoSQL database for storing and retrieving dynamic data.
  • AWS API Gateway:reciever of of data from website and forwarding it to lambda
