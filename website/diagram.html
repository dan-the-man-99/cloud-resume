How this AWS Website Works:

This website works by using AWS services. It utilizes HTML, CSS, and JavaScript for the frontend. The backend infrastructure is powered by a combination of AWS services:

  • Route 53: Provides domain registration and DNS routing.
  • ACM (AWS Certificate Manager): Manages SSL/TLS certificates for secure communication.
  • CloudFront: Acts as a content delivery network (CDN) to accelerate content delivery globally.
  • S3 (Simple Storage Service): Stores static assets such as HTML files, JavaScript files, CSS stylesheets, and images.
  • Lambda: Executes serverless functions in this case for retrieves the value from the dynamoDB increment it by 1 prints the new value then update the dynamoDB count
  • DynamoDB: Provides a NoSQL database for storing and retrieving dynamic data.

I have also created a GitHub repository for this project. However, CI/CD (Continuous Integration/Continuous Deployment) has not been implemented yet. I plan to use Terraform for automating the deployment process, so stay tuned for updates!
